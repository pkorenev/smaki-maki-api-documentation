Products  ¶

Get products
GET/products

if provided `product_ids` 
  returns only products with `id` found in provided `product_ids` 
else if provided `category_id` 
  returns only products related category.
else if provided `subcategory_id` 
  returns only products related to category.
else if both `category_id` and `subcategory_id` provided
  if subcategory related to provided category
    returns products related to subcategory.
  else
    returns empty array.
else
  returns all products.

module LastUpdateDateTime

  • Parameters:
    • last_update_datetime - datetime

returns only resources created/updated since provided last_update_datetime.
